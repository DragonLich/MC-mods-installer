1. stáhnout forge installer z http://files.minecraftforge.net/ pro MC 1.7.10 verzi recommended
2. nainstalovat do své MC verze
3. spustit MC s vytvořeným profilem Forge (pokud již není vytvořen tak vytvořit a nastavit verzi na release *-Forge*
4. vypnout MC
5. rozbalit balík módů a módy nakopírovat do složky .minecraft (linux ~/.minecraft/mods   ||   %appdata%\.minecraft\mods)
6. spustit MC
7. hrát :-)
